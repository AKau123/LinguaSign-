# 🧠 Linguasign: Smart Gesture-to-Speech & Speech-to-Text System

## 📌 Overview
Linguasign is an assistive communication system designed to bridge the gap between speech and hearing impaired individuals and the rest of society. The system converts hand gestures into speech and also enables speech-to-text conversion for two-way communication.

## 🚨 Problem Statement
Communication barriers faced by deaf and mute individuals limit their ability to interact effectively in daily life. Existing solutions are often expensive or lack real-time responsiveness.

## 🎯 Objective
To build a cost-effective, real-time embedded system that enables:
- Gesture → Speech conversion
- Speech → Text conversion

## 🛠️ Tech Stack
- **Hardware:** Arduino, Sensors (Gesture/Input)
- **Software:** Embedded C / Arduino IDE
- **Concepts:** Signal Processing, Real-Time Systems, Assistive Technology

## ⚙️ System Architecture
1. Gesture input captured via sensors  
2. Processed using embedded logic  
3. Converted into predefined speech output  
4. Speech input captured and converted into text  

## 🔍 Methodology
- Implemented gesture recognition using sensor-based inputs  
- Integrated speech synthesis module for audio output  
- Used speech-to-text processing for reverse communication  
- Designed a real-time pipeline ensuring minimal delay  

## 📊 Results
- Achieved **~90–93% accuracy** in gesture recognition and communication  
- Enabled **real-time bidirectional communication**  
- Demonstrated stable performance with low latency  

## 🌍 Impact
This project contributes towards:
- Inclusive communication technology  
- Assistive solutions for differently-abled individuals  
- Affordable real-time embedded systems  

## 🚀 Future Improvements
- Integration with Machine Learning for dynamic gesture recognition  
- Mobile app connectivity (Bluetooth/WiFi)  
- Multi-language speech support
